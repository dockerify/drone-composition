### drone-composition

#### usage

- create `.drone.env` file with the following environment variables


```
DRONE_OPEN=false
DRONE_HOST=http://localhost:8000
DRONE_ORGS=
DRONE_ADMIN=
DRONE_GITHUB=true
DRONE_GITHUB_CLIENT=
DRONE_GITHUB_SECRET=
```

- create `.drone.secret.env` file with the following environment variables

```
DRONE_SECRET=
```
